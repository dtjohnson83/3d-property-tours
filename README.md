# 3D Property Tours - Real Estate Tool
**Status:** Prototype
**Cost:** ~$1.20 per property (Standard) or $0.12 (Draft)
**Charge:** $25-50 per listing

## How It Works
1. Agent takes photos of property (16 photos for full 360° or multiple angles per room)
2. Script uploads to World Labs API
3. API returns navigable 3D environment
4. Shareable link sent to agent/buyer

## Setup
1. Get API key from platform.worldlabs.ai
2. Save to `credentials/worldlabs-credentials.json`
3. Buy minimum credits ($5 = 6,250 credits)
4. Run script
